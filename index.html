    Sobre o Challenge One


    Este projeto faz parte do Projeto ONE desenvolvido pela Oracle Next Education e Alura Latam


    Esse é o Challenge da Trilha - Iniciante em Programação que tem como objetivo o desenvolvimento de um codificador de textos com a utilização das tecnologias HTML, CSS e JavaScript, além de nos fornecer conhecimentos sobre a utilização do Trello como ferramenta de acompanhamento de desenvolvimento e utilização do Git/ Github para o versionamneto de código.
